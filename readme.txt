Connect AutomatorWP with WishList Member

== Description ==

> **Important:** Now all free integrations are included in [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") so you no longer need to install them one by one!
>
> We will continue working to improve AutomatorWP and make it easier, faster and more accessible to everyone.

[WishList Member](https://member.wishlistproducts.com/ "WishList Member") is a powerful, yet easy to use membership software solution that can turn any WordPress site into a full-blown membership site.

This plugin automatically connects [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") with WishList Member adding new triggers and actions that you can use to connect with other plugins and automate workflows letting you save time and get focused on your most important work.

= Triggers =

* User gets added to any/specific level.
* User gets removed from any/specific level.

= Pro version =

[Pro version](https://automatorwp.com/add-ons/wishlist-member/ "AutomatorWP - WishList Member") available that let's you expand the integration between AutomatorWP and WishList Member bringing you more ways to automatize your membership website.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
